Fates Hub is the continuation of the Exodus experience, where players embark on a journey to their destination asteroid.

As the lead developer, I created the experience in Unity and collaborated closely with both the creative and backend teams to ensure a seamless and engaging interactive experience.

Character Viewer

I developed a character package that optimizes character-specific addressables, combining meshes and animations in real-time for optimal performance.







Scheduled Events

I utilized Unity Remote Config and Game Overrides to schedule unique events and unlock features through JSON data. One example was a dynamic storm event that evolved over several weeks.

Games

I developed a modular, stateful games system that can be toggled using JSON data from Unity Remote Config. This game was designed to record player progress over a 7-day period, receiving and sending data to a server. As players progress, they earn experience and level up.
